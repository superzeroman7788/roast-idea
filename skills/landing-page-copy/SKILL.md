# Landing Page Copy

## Purpose

Write conversion-focused Chinese copy for digital products. Chinese consumer copy has distinct patterns — it is more direct about benefits, uses social proof differently, and often leads with emotional resonance before features.

## Input

Accept: product brief, direction card, target user description, or existing rough copy to improve. If given auto-pilot output, use `direction`, `open_questions`, and `artifacts_hint` fields.

## Output Structure

Produce all sections unless user specifies fewer:

```
## Hero Section
**主标题**（≤12字，核心利益点，不是产品名）
**副标题**（≤25字，补充主标题，点出目标用户+场景）
**CTA 文案**（≤6字，动作明确）

## 三大价值点
1. **[价值标题]** — [一句话解释，≤30字]
2. ...
3. ...

## 用户证言钩子（模板）
「[具体场景中的真实感受，引号内，第一人称]」
— [用户身份，不用全名，用角色：22岁女大学生 / 深圳创业者]

## 小红书/朋友圈文案变体
[同一产品的社交媒体版本，加上 3 个话题标签]

## App Store 简介（≤80字）
[直接可用版本]
```

## Writing Rules

**主标题规则：**
- 聚焦结果，不聚焦功能：「失恋后，重新认识自己」而不是「情绪记录 App」
- 用数字制造具体感：「7天找到你的核心竞争力」
- 禁止：「智能」「AI赋能」「全方位」「专业」等烂大街形容词

**中文消费品文案的核心公式：**
`[用户当前痛苦/场景] + [我们做到的转变] + [具体证明]`

**情绪节奏：**
先共情（你也有这个问题），再给希望（可以这样解决），再给证明（别人已经做到了），最后给行动（现在就开始）。

**女性消费品额外规则（如适用）：**
- 避免说教语气，多用「发现」「找到」「变成」等自主词汇
- 截图分享文化：文案要能出现在小红书截图里还显得好看
- 用「你」不用「您」，用「我们」让用户感觉是自己的选择

## Quality Check

- 主标题能在 3 秒内让目标用户说「这说的是我」
- 没有一句话在描述技术而不是用户感受
- CTA 文案说的是用户得到什么，不是让用户「注册」「下载」
